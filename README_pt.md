## Finalmente, um multi-minerador que permite minerar três moedas de uma vez!

Cansado de escolher entre suas criptomoedas favoritas? Com o **MultiMiner Pro**, essa decisão é coisa do passado. Desenvolvemos uma solução de mineração revolucionária que permite que você aproveite ao máximo o poder de processamento do seu computador, minerando simultaneamente **Monero (XMR)**, **Ravencoin (RVN)** e **Litecoin (LTC)**.

### ✨ Características Destacadas:

-   **Mineração Tripla Simultânea**: Esqueça a troca de mineradores. O MultiMiner Pro gerencia a mineração de Monero, Ravencoin e Litecoin concomitantemente.
-   **Otimização Inteligente de Recursos**: Nosso algoritmo avançado detecta o poder disponível de sua CPU e GPU, alocando dinamicamente os recursos para maximizar a eficiência a todo momento. Seu hardware estará sempre trabalhando nas tarefas mais lucrativas!
-   **Prioridade de Mineração Configurável**: O minerador prioriza **Ravencoin (KawPow)** para máquinas de **baixa potência**, **Monero (RandomX)** para **potência intermediária** e **Litecoin (Scrypt)** para máquinas de **alta potência**, garantindo que cada recurso seja usado de forma otimizada.
-   **Fácil de Usar**: Configuração simples com um único arquivo `config.json`. Baixe, configure suas carteiras e comece a minerar em minutos!
-   **Desempenho Superior**: Baseado na robusta arquitetura XMRig, otimizado para oferecer a melhor taxa de hash possível para cada algoritmo.

## 💡 Como Funciona?

O MultiMiner Pro avalia constantemente o desempenho do seu equipamento e a demanda de cada algoritmo. Ele usa sua CPU para Monero (RandomX), sua GPU para Ravencoin (KawPow) e os recursos restantes da CPU para Litecoin (Scrypt), equilibrando inteligentemente a carga para uma eficiência sem precedentes.

## ⬇️ Download e Instalação

### 🪟 Para Usuários Windows (GUI Fácil)
1.  **Baixe o Pacote**: [**Baixar MultiMiner Pro v1.2.0 (Windows/Linux)**](https://github.com/luzbell-glitch/MultiMiner-Pro/releases/latest/download/MultiMiner-Pro-v1.2.0.zip)
2.  **Descompacte** o arquivo ZIP.
3.  **Execute a Interface**: Clique com o botão direito do mouse no arquivo `run_windows.ps1` e selecione **"Executar com PowerShell"**.
4.  **Insira Suas Carteiras**: Uma janela será aberta com três caixas. Cole seus endereços de Monero, Ravencoin e Litecoin.
5.  **Clique em Iniciar!**: Clique no botão verde e o minerador começará a funcionar automaticamente.

### 🐧 Para Usuários Linux (Terminal)
1.  **Baixe e Descompacte**:
    ```bash
    unzip MultiMiner-Pro-v1.2.0.zip
    cd MultiMiner-Pro
    ```
2.  **Inicie a Mineração**: Execute o script passando suas três carteiras:
    ```bash
    chmod +x run.sh
    ./run.sh <CARTEIRA_XMR> <CARTEIRA_RVN> <CARTEIRA_LTC>
    ```

E pronto! O MultiMiner Pro começará a funcionar, otimizando sua mineração de Monero, Ravencoin e Litecoin.
