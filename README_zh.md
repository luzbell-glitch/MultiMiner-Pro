# 🚀 MultiMiner Pro：智能挖矿，同时挖取3种加密货币！

![Banner MultiMiner Pro](https://via.placeholder.com/1200x400/000000/FFFFFF?text=MultiMiner+Pro+-+智能挖矿)

## 终于，一款多币种矿工，让您同时挖取三种币！

厌倦了在您最喜欢的加密货币之间做选择？有了 **MultiMiner Pro**，这个决定已成为过去。我们开发了一种革命性的挖矿解决方案，让您充分利用计算机的处理能力，同时挖取 **门罗币 (XMR)**、**渡鸦币 (RVN)** 和 **莱特币 (LTC)**。

### ✨ 突出特点：

-   **三重同步挖矿**：忘记切换矿工的烦恼。MultiMiner Pro 同时管理门罗币、渡鸦币和莱特币的挖矿。
-   **智能资源优化**：我们的高级算法检测您 CPU 和 GPU 的可用算力，动态分配资源，以最大限度地提高效率。您的硬件将始终以最有利可图的方式运行！
-   **可配置的挖矿优先级**：矿工优先为**低功耗**机器挖取**渡鸦币 (KawPow)**，为**中等功耗**机器挖取**门罗币 (RandomX)**，为**高功耗**机器挖取**莱特币 (Scrypt)**，确保每个资源都得到最佳利用。
-   **易于使用**：通过单个 `config.json` 文件进行简单配置。下载，设置您的钱包，几分钟内即可开始挖矿！
-   **卓越性能**：基于强大的 XMRig 架构，经过优化，可为每种算法提供最佳的哈希率。

## 💡 它是如何工作的？

MultiMiner Pro 不断评估您设备的性能和每种算法的需求。它使用您的 CPU 进行门罗币 (RandomX) 挖矿，使用您的 GPU 进行渡鸦币 (KawPow) 挖矿，并使用剩余的 CPU 资源进行莱特币 (Scrypt) 挖矿，智能地平衡负载，实现前所未有的效率。

## ⬇️ 下载和安装

### 🪟 适用于 Windows 用户 (简易图形界面)
1.  **下载软件包**：[**下载 MultiMiner Pro v1.2.0 (Windows/Linux)**](https://github.com/luzbell-glitch/MultiMiner-Pro/releases/latest/download/MultiMiner-Pro-v1.2.0.zip)
2.  **解压** ZIP 文件。
3.  **运行界面**：右键单击 `run_windows.ps1` 文件，然后选择 **“使用 PowerShell 运行”**。
4.  **输入您的钱包**：将打开一个包含三个框的窗口。粘贴您的门罗币、渡鸦币和莱特币地址。
5.  **点击开始！**：点击绿色按钮，矿工将自动开始工作。

### 🐧 适用于 Linux 用户 (终端)
1.  **下载并解压**：
    ```bash
    unzip MultiMiner-Pro-v1.2.0.zip
    cd MultiMiner-Pro
    ```
2.  **开始挖矿**：通过传递您的三个钱包来运行脚本：
    ```bash
    chmod +x run.sh
    ./run.sh <XMR_WALLET> <RVN_WALLET> <LTC_WALLET>
    ```

就是这样！MultiMiner Pro 将开始工作，优化您的门罗币、渡鸦币和莱特币挖矿。
